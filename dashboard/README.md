# Dashboard

Tableau is the first dashboard target.

Use the exported CSV files from:

```text
data/processed/dashboard_exports/
```

Recommended Tableau pages:

- Executive Overview
- Sales Trends
- Product Performance
- Customer Analysis
- Returns and Quality

Add screenshots to:

```text
dashboard/screenshots/
```

Current preview:

- `screenshots/executive_overview_preview.svg`

Power BI can be added later under:

```text
dashboard/powerbi/
```
